# Roteiro Oficial de Entrevista — Empresa Júnior
## Documento de referência

Versão 0.1 — Julho de 2026

---

Este é o roteiro-base usado como ponto de partida em qualquer entrevista inicial de projeto na Empresa Júnior. Cada projeto deve adaptar as perguntas ao seu contexto — este documento existe para não começar do zero toda vez, e para que melhorias no roteiro beneficiem todos os projetos futuros sem precisar editar cada um individualmente.

## Perguntas-base

**Sobre o problema**
- Qual problema esse projeto deve resolver?
- Por que esse problema importa agora?
- O que acontece hoje, na ausência de uma solução?

**Sobre quem usa**
- Quem vai usar o resultado deste projeto?
- Existe alguma característica importante desse público (idade, experiência prévia, contexto)?

**Sobre o formato**
- Existe alguma preferência ou restrição de formato (plataforma, tecnologia, estilo)?
- Existe algo que definitivamente NÃO deve fazer parte do escopo?

**Sobre prazo e expectativas**
- Existe um prazo desejado?
- Como será medido se o projeto deu certo?

## Como usar

1. Antes da entrevista, quem for conduzi-la deve estudar este roteiro (missão "Estudar modelo de entrevista").
2. Adaptar as perguntas ao contexto específico do projeto, gerando um roteiro próprio (missão "Montar roteiro de entrevista").
3. Conduzir a entrevista de fato (missão "Entrevistar responsável").

Este roteiro deve evoluir conforme a Empresa Júnior aprender, na prática, quais perguntas geram melhores respostas.
